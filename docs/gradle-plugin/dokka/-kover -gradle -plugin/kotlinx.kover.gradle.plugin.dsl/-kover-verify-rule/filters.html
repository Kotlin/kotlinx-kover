filters

abstract fun filters(config: Action<KoverReportFilters>)

Specifies the set of Kover report filters that control included or excluded classes and packages for verification.

An example of filter configuration:

filters {
   excludes {
       // Do not include deprecated package into verification coverage data
       className("org.jetbrains.deprecated.*")
    }
}
